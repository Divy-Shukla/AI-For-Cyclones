# AI-For-Cyclones

## Cyclone Early Warning System

A React + Tailwind CSS web application providing an AI-powered minimal, futuristic UI for real-time tracking, prediction, and alerts for cyclones.

### Features
- Live Cyclone Map via CartoDB & Leaflet.
- AI Trend Predictions with Recharts.
- Geo-spatial alert system preferences setup.
- Regional Heatmap risk assessment.

### Setup
```bash
npm install
npm run dev
```
